Compilation / Run Instructions
-------------------------------

 1. Open a terminal (Command Prompt or PowerShell on Windows).

  2. Navigate to the src/ directory:
       cd path\to\your\project\src

     Example:
       cd C:\Users\YourName\Desktop\java\v3\src

  3. Compile all Java files in one command:
       javac *.java

  4. If compilation is successful, you will see no output and .class files
     will appear in the same directory.

     If you see errors, make sure:
       - All .java files are in the same flat src/ directory (no subfolders).
       - You are using Java 17 or higher.
    

  5. After successful compilation, run the application with:

    java Main

  The application window will open automatically (950 × 720 px).

  Full example (compile + run):
    cd C:\Users\YourName\Desktop\java\v3\src
    javac *.java
    java Main
